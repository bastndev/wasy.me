1. download or clone project

  ```sh
  https://github.com/bastndev/wasy.me.git
  ```

2. install package

  ```sh
  npm install install
  ```

- o NPM

  ```sh
  npm install i
  ```

3. start project

   ```sh
   pnpm run dev
   ```
